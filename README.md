## Nota

Esta lista se centra en software open source que da **soporte específico a Catalunya** o que incluye funcionalidades relevantes para usuarios en esta comunidad autónoma. No se incluye software genérico solo por estar creado por desarrolladores catalanes.

## Descargo de responsabilidad

No se aceptan proyectos relacionados con pornografía, contenido NSFW, loterías o apuestas, religión, política partidista ni cualquier otro tema controvertido. Esta lista pretende ser un recurso técnico neutral y útil para la comunidad de desarrolladores.
